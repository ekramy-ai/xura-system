# نظام الحكام والصلاحيات الجديد — XURA

تم الانتهاء من بناء نظام متكامل لإدارة الحكام وصلاحيات الدخول للمباريات عبر Firebase.

## 1. نظام المسؤول (Admin)
- **صفحة الدخول:** [xura-admin-login.html](http://localhost:8080/xura-admin-login.html)
- **رمز الدخول الافتراضي:** `0000`
- **الحماية:** لا يمكن الدخول لصفحة "إدارة قاعدة البيانات" أو "إعداد المباراة" بدون تسجيل دخول المسؤول.
- **إدارة الحكام:** يمكنك الآن إضافة حكام جدد وتحديد PIN Code خاص لكل حكم من صفحة [xura-db-manager.html](http://localhost:8080/xura-db-manager.html).

## 2. نظام الحكام (Referee)
- **صفحة الدخول:** [xura-referee-login.html](http://localhost:8080/xura-referee-login.html)
- **طريقة العمل:** 
  1. يختار الحكم اسمه من القائمة.
  2. يدخل الـ PIN Code الخاص به.
  3. تظهر له قائمة **المباريات المعينة له فقط**.
- **الصلاحيات:** إذا حاول حكم الدخول لمباراة غير معينة له عبر الرابط، سيتم منمنعه وتوجيهه لصفحة الدخول.

## 3. إعداد المباراة (Match Setup)
- عند إنشاء مباراة جديدة، يجب اختيار **الحكم المعين** من القائمة المنسدلة.
- يتم حفظ اسم الحكم وهويته مع بيانات المباراة لضمان الصلاحيات.

## 4. لوحة النتائج المباشرة (Live Dashboard)
- **الرابط:** [xura-live-v2.html](http://localhost:8080/xura-live-v2.html)
- **التطوير:** اللوحة الآن تدعم عرض **أكثر من مباراة حية في نفس الوقت**.
- **بيانات إضافية:** يظهر اسم الحكم المسؤول عن كل مباراة على بطاقة النتائج.

---

### كيفية التجربة:
1. ادخل كمسؤول (`0000`) وافتح [إدارة قاعدة البيانات](http://localhost:8080/xura-db-manager.html).
2. أضف حكماً جديداً (مثلاً: "أحمد" برمز `1234`).
3. اذهب لصفحة [إعداد المباراة](http://localhost:8080/match-setup.html) وأنشئ مباراة وقم بتعيين "أحمد" لها.
4. افتح صفحة [دخول الحكم](http://localhost:8080/xura-referee-login.html) وسجل دخول بـ "أحمد" ورمز `1234`.
5. ابدأ المباراة وشاهد النتائج تظهر فوراً على [اللوحة المباشرة](http://localhost:8080/xura-live-v2.html).
